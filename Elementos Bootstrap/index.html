Tables

Para trabajar con tablas en Bootstrap 5, es fundamental utilizar la clase base table, que aplica estilos básicos como bordes suaves y una alineación consistente.

Para mejorar la apariencia visual, puedes añadir clases como table-striped para alternar el fondo de las filas, table-bordered para agregar bordes a todas las celdas, y table-hover para resaltar las filas al pasar el cursor sobre ellas. Si prefieres un esquema de colores oscuro, la clase table-dark aplica un fondo oscuro a la tabla.

Para hacer que una tabla sea responsive y se adapte a diferentes tamaños de pantalla, envuélvela dentro de un contenedor con la clase table-responsive. Esto permite que la tabla se desplace horizontalmente en dispositivos con pantallas más pequeñas.

Además, puedes utilizar clases como table-sm para reducir el espaciado de las celdas y hacer la tabla más compacta. Para alinear el contenido de las celdas, emplea clases como text-start, text-center y text-end. Si deseas aplicar colores específicos a filas o celdas, puedes usar clases como table-primary, table-success o table-danger, que aplican colores de fondo predefinidos para resaltar información importante.

Para personalizar el encabezado de la tabla, las clases thead-dark y thead-light permiten aplicar estilos oscuros o claros al fondo del elemento <thead>, diferenciándolo visualmente del resto de la tabla.

Dropdowns

Un menu dropdown es un menú alternable que permite al usuario elegir un valor de una lista predefinida.

Se crea con la clase dropdown, para abrir el menú desplegable, utilice un botón o un enlace con la clase .dropdown-toggle y el atributo data-bs-toggle="dropdown".

Agregue la .dropdown-menu clase a un elemento contenedor, como <div> o <ul>, para crear el menú desplegable. Luego, agregue la .dropdown-item clase a cada elemento (enlaces o botones) dentro del menú desplegable.

También se le pueden añadir divisores para separar elementos, con la clase .dropdown-divider. Como también podemos agregar encabezados para agrupar los elementos, esta opción se hace con la clase .dropdown-header.

Forms

Los formularios en Bootstrap 5 son un conjunto de clases y estilos que permiten construir formularios responsivos, estéticos y fáciles de usar. Incluyen soporte para una variedad de elementos como entradas de texto, botones, checkboxes, radios y más.

Utiliza la clase .form-control para campos de entrada como <input> o <textarea>, y .form-select para listas desplegables.

Los formularios también pueden estructurarse utilizando clases como .row y .col para crear distribuciones en línea o apiladas.

Además, se pueden personalizar con tamaños mediante clases como .form-control-lg y .form-control-sm, y se pueden agregar estilos interactivos para checkboxes y radios con .form-check, .form-check-input y .form-check-label.

Tooltip

Los tooltips son pequeños cuadros de texto que aparecen al pasar el cursor sobre un elemento. Bootstrap permite implementarlos fácilmente utilizando el atributo data-bs-toggle="tooltip".

Activar Tooltips

Para habilitar tooltips, debes incluir el atributo data-bs-toggle="tooltip" en un elemento y asignarle un mensaje utilizando el atributo title. Asegúrate de inicializar los tooltips con JavaScript.

Nota: Es necesario incluir el paquete de JavaScript de Bootstrap para que los tooltips funcionen.



Posición del Tooltip

Puedes cambiar la posición del tooltip utilizando el atributo data-bs-placement. Las opciones disponibles son: top, right, bottom, y left.


Tooltip Activado Automáticamente

Los tooltips no se activan automáticamente; necesitas inicializarlos manualmente con JavaScript. Aquí tienes un ejemplo del código necesario:

      
        // Inicializar todos los tooltips en la página
        var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'));
        var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
          return new bootstrap.Tooltip(tooltipTriggerEl);
        });

Range

Los deslizadores (Range) en Bootstrap son controles de entrada que permiten seleccionar un valor dentro de un rango definido. Se crean con el elemento <input type="range"> y se mejoran visualmente con las clases de Bootstrap.

Básico

Para crear un control deslizante básico, utiliza el elemento <input type="range"> con la clase .form-range. No requiere un contenedor adicional.


Con Rango Mínimo y Máximo

Puedes definir el rango mínimo y máximo del deslizador utilizando los atributos min y max. También puedes establecer un paso con el atributo step.


Deslizador Deshabilitado

Un deslizador puede deshabilitarse agregando el atributo disabled. Esto es útil para evitar que el usuario interactúe con el control en ciertas condiciones.


Inicialización con JavaScript

Puedes controlar dinámicamente el valor del deslizador utilizando JavaScript. Esto te permite actualizar el rango en tiempo real o sincronizarlo con otras interacciones de usuario.


// Ejemplo: Cambiar el valor del deslizador dinámicamente
document.getElementById('customRange').value = 50;

Scrollspy

Scrollspy permite sincronizar una barra de navegación con las secciones de la página, resaltando automáticamente el enlace activo según la posición del usuario al desplazarse.

Añade data-bs-spy="scroll" al elemento que debe usarse como área desplazable (a menudo es este el <body> elemento).

A continuación, agrega el data-bs-target atributo con un valor de id o el nombre de la clase de la barra de navegación (.navbar). Esto es para asegurarse de que la barra de navegación esté conectada con el área desplazable.

Tenga en cuenta que los elementos desplazables deben coincidir con el ID de los enlaces dentro de los elementos de lista de la barra de navegación (<div id="section1"> coincidencias <a href="#section1">).

El atributo opcional data-bs-offset especifica la cantidad de píxeles que se deben desplazar desde la parte superior al calcular la posición de desplazamiento. Esto resulta útil cuando cree que los enlaces dentro de la barra de navegación cambian de estado activo demasiado pronto o demasiado pronto al saltar a los elementos desplazables. El valor predeterminado es 10 píxeles.

Popover

Un Popover es un componente que muestra contenido adicional cuando el usuario interactúa con un elemento, generalmente al hacer clic o pasar el cursor por encima de un botón o enlace. A diferencia de los tooltips, los popovers pueden contener texto y otros elementos como encabezados, botones, y listas.

Ejemplo básico

Para crear un popover básico, necesitamos añadir el atributo data-bs-toggle="popover" al elemento de activación y luego usar el atributo title y data-bs-content para definir el contenido del popover.


Activación de popover con JavaScript

También podemos inicializar los popovers utilizando JavaScript. Para hacer esto, necesitamos usar el método Popover de Bootstrap.


Popover con varias configuraciones

Los popovers tienen varias configuraciones que se pueden personalizar:

  • placement: Define la posición del popover. Puede ser top, bottom, left, o right.
  • trigger: Define el tipo de activación. Los valores comunes son click, hover, y focus.

Floating Labels

Floating Labels son etiquetas que flotan sobre los campos de formulario, lo que mejora la experiencia del usuario al mostrar el texto de la etiqueta dentro del campo hasta que el usuario interactúa con el campo. Luego, la etiqueta flota hacia arriba, dejando el espacio disponible para el texto ingresado.

Ejemplo básico de Floating Label

Para usar las etiquetas flotantes en los formularios, debes agregar las clases .form-floating al contenedor del campo de formulario, y la etiqueta <label> debe tener el atributo for que coincide con el id del campo de entrada.


Floating Labels en un formulario con múltiples campos

Se puede utilizar esta funcionalidad en varios campos dentro de un formulario para crear una interfaz más limpia y moderna.


Floating Labels con validación

Las etiquetas flotantes también se pueden usar con la validación de formularios. Puedes agregar clases de validación como .is-valid o .is-invalid a los campos de entrada.

Checkboxes y Radio Buttons

Checkboxes y Radio Buttons son elementos de formulario utilizados para seleccionar opciones. Los checkboxes permiten seleccionar varias opciones, mientras que los radio buttons permiten seleccionar solo una opción entre varias.

Los Checkboxes se utilizan cuando el usuario puede seleccionar más de una opción. Para implementar un checkbox, simplemente utilizamos el elemento <input type="checkbox">.


Los Radio Buttons se utilizan cuando el usuario solo puede seleccionar una opción de varias disponibles. Para implementar un radio button, utilizamos el elemento <input type="radio">.


Checkboxes y Radio Buttons con validación

Se pueden aplicar clases de validación como .is-invalid o .is-valid a los checkboxes y radio buttons para mostrar el estado de validación.

En Bootstrap 5, el componente Jumbotron ha sido descontinuado. Sin embargo, puedes crear un efecto similar usando utilidades de clases, como mt-4, p-5, bg-primary, text-white, y rounded.

Explicación del código:

  • mt-4: Añade un margen superior de 4 unidades, separando el Jumbotron de lo que esté por encima de él.
  • p-5: Aplica un padding de 5 unidades alrededor del contenido dentro del Jumbotron, dándole un espacio adecuado.
  • bg-primary: Establece el fondo del Jumbotron en el color principal de Bootstrap (azul en este caso), pero puede cambiarse a otros colores como bg-secondary, bg-danger, etc.
  • text-white: Establece el color del texto en blanco para que contraste adecuadamente con el fondo.
  • rounded: Aplica bordes redondeados al Jumbotron para un aspecto más moderno y estilizado.

Dentro del contenedor, también se incluye:

  • display-4: Un estilo de encabezado grande que resalta el título del Jumbotron.
  • lead: Una clase de texto que aplica un estilo más ligero y destacado al párrafo de descripción.
  • btn btn-light btn-lg: Un botón de gran tamaño con fondo claro, que permite a los usuarios explorar más contenido.
