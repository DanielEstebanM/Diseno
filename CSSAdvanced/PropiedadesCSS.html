CSS Border Images

Con la propiedad CSS border-image, puedes configurar una imagen para que se utilice como borde alrededor de un elemento.

Propiedad CSS border-image

La propiedad CSS border-image le permite especificar una imagen que se utilizará en lugar del borde normal alrededor de un elemento.

La propiedad tiene tres partes:

  1. La imagen que se utilizará como borde
  2. Dónde cortar la imagen
  3. Definir si las secciones intermedias deben repetirse o estirarse

La propiedad border-image toma una imagen y la divide en nueve secciones, como un tablero de tres en raya. Luego, coloca las esquinas en las esquinas y las secciones del medio se repiten o se estiran según lo especifique.

Aquí, las secciones centrales de la imagen se repiten para crear el borde:

border-image: url(imagen.png) 30 round

Aquí, las secciones centrales de la imagen se estiran para crear el borde:

border-image: url(imagen.png) 30 strech

Los diferentes valores de corte cambian por completo el aspecto del borde:

Ejemplo 1:

border-image: url(imagen.png) 50 round

Los diferentes valores de corte cambian por completo el aspecto del borde:

Ejemplo 2:

border-image: url(imagen.png) 20% round

Los diferentes valores de corte cambian por completo el aspecto del borde:

Ejemplo 3:

border-image: url(imagen.png) 30% round

CSS Multiple Backgrounds

En este capítulo aprenderá cómo agregar varias imágenes de fondo a un elemento.

También aprenderá sobre las siguientes propiedades:

  • background-size
  • background-origin
  • background-clip

CSS le permite agregar múltiples imágenes de fondo para un elemento, a través de la propiedad background-image.

Las diferentes imágenes de fondo están separadas por comas y las imágenes se apilan una encima de otra, donde la primera imagen es la más cercana al espectador.

Tamaño (background-size)

La propiedad background-size le permite especificar el tamaño de las imágenes de fondo.

El tamaño se puede especificar en longitudes, porcentajes o utilizando una de las dos palabras clave: content o cover.

El siguiente ejemplo cambia el tamaño de una imagen de fondo a un tamaño mucho más pequeño que la imagen original (usando píxeles):

background: url(img_fondo.jpg);

background-size: 100px 80px;

background-repeat: no-repeat;

Propiedad CSS background-origin

La propiedad CSS background-origin especifica dónde se posiciona la imagen de fondo.

La propiedad toma tres valores diferentes:

  1. border-box: la imagen de fondo comienza desde la esquina superior izquierda del borde
  2. padding-box - (predeterminado) la imagen de fondo comienza desde la esquina superior izquierda del borde de relleno
  3. content-box: la imagen de fondo comienza desde la esquina superior izquierda del contenido

background: url(img_fondo.jpg);

background-repeat: no-repeat;

background-origin: padding-box;

background: url(img_fondo.jpg);

background-repeat: no-repeat;

background-origin: border-box;

background: url(img_fondo.jpg);

background-repeat: no-repeat;

background-origin: content-box;

Propiedad CSS background-clip

La propiedad CSS background-clip especifica el área de pintura del fondo.

La propiedad toma tres valores diferentes:

  1. border-box - (predeterminado) el fondo se pinta hasta el borde exterior del borde
  2. padding-box: el fondo se pinta hasta el borde exterior del relleno.
  3. content-box: el fondo se pinta dentro del cuadro de contenido

border: 10px dotted black;

padding: 35px

background: yellow;

background-clip: border-box;

border: 10px dotted black;

padding: 35px

background: yellow;

background-clip: padding-box;

border: 10px dotted black;

padding: 35px

background: yellow;

background-clip: content-box;

CSS Colors

En esta sección veremos los siguientes valores:

  • Colores RGBA
  • Colores HSL
  • Colores HSLA
  • Opacidad

Colores RGBA

Los valores de color RGBA son una extensión de los valores de color RGB con un canal alfa, que especifica la opacidad de un color.

Un valor de color RGBA se especifica con: rgba(rojo, verde, azul, alfa). El parámetro alfa es un número entre 0,0 (totalmente transparente) y 1,0 (totalmente opaco).

rgba(255, 0, 0, 0.2);

rgba(255, 0, 0, 0.4);

rgba(255, 0, 0, 0.6);

rgba(255, 0, 0, 0.8);

Colores HSL

HSL significa Tono, Saturación y Luminosidad.

Un valor de color HSL se especifica con: hsl(tono, saturación, luminosidad).

  • El tono es un grado en la rueda de color (de 0 a 360):
    1. 0 (o 360) es rojo
    2. 120 es verde
    3. 240 es azul
  • La saturación es un valor porcentual: 100% es el color completo.
  • La luminosidad también es un porcentaje: 0% es oscuro (negro) y 100% es blanco.

hsl(133, 50%, 30%);

hsl(133, 50%, 50%);

hsl(133, 50%, 70%);

hsl(133, 50%, 90%);

Colores HSLA

Los valores de color HSLA son una extensión de los valores de color HSL con un canal alfa, que especifica la opacidad de un color.

Un valor de color HSLA se especifica con: hsla(tono, saturación, luminosidad, alfa), donde el parámetro alfa define la opacidad. El parámetro alfa es un número entre 0,0 (totalmente transparente) y 1,0 (totalmente opaco).

hsla(133, 50%, 30%, 0.5);

hsla(133, 50%, 50%, 0.5);

hsla(133, 50%, 70%, 0.5);

hsla(133, 50%, 90%, 0.5);

Opacidad

La propiedad CSS opacity establece la opacidad de todo el elemento (tanto el color de fondo como el texto serán opacos/transparentes).

El valor de la propiedad debe ser un número entre 0,0 (totalmente transparente) y 1,0 (totalmente opaco).

rgb(3, 170, 0);

opacity: 0.2;

rgb(3, 170, 0);

opacity: 0.4;

rgb(3, 170, 0);

opacity: 0.6;

rgb(3, 170, 0);

opacity: 0.8;

¡Tenga en cuenta que el texto de arriba también será transparente/opaco!

CSS Color Keywords

Esta parte explicará las palabras clave transparent, currentcolor, e inherit.

Transparent

La palabra clave transparent se utiliza para hacer que un color sea transparente. Esto se suele utilizar para hacer que el color de fondo de un elemento sea transparente.

background-color: transparent;

Nota: La palabra clave transparent es equivalente a rgba(0,0,0,0). Los valores de color RGBA son una extensión de los valores de color RGB con un canal alfa, que especifica la opacidad de un color. Obtenga más información en nuestro capítulo CSS RGB y en nuestro capítulo Colores CSS.

Currentcolor

La palabra clave currentcolor es como una variable que contiene el valor actual de la propiedad de color de un elemento.

Esta palabra clave puede ser útil si desea que un color específico sea consistente en un elemento o una página.

color: rgb(88, 140, 237);

border: 10px solid currentcolor;

Inherit

La palabra clave inherit especifica que una propiedad debe heredar su valor de su elemento padre.

Esta palabra clave se puede utilizar para cualquier propiedad CSS y en cualquier elemento HTML.

border: inherit;

CSS Gradients

Los degradados CSS le permiten mostrar transiciones suaves entre dos o más colores específicos.

CSS define tres tipos de degradados:

  • Degradados lineales (hacia abajo/arriba/izquierda/derecha/ diagonalmente)
  • Gradientes radiales (definidos por su centro)
  • Gradientes cónicos (rotados alrededor de un punto central)

Degradados lineales

Para crear un degradado lineal, debes definir al menos dos niveles de color. Los niveles de color son los colores entre los que quieres crear transiciones suaves. También puedes establecer un punto de inicio y una dirección (o un ángulo) junto con el efecto de degradado.

Sintaxis

background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

Dirección: de arriba a abajo (este es el valor predeterminado)

El siguiente ejemplo muestra un degradado lineal que comienza en la parte superior. Comienza en rojo y pasa al amarillo:

background-image: linear-gradient(red, yellow);

Dirección - De izquierda a derecha

El siguiente ejemplo muestra un degradado lineal que comienza desde la izquierda. Comienza en rojo y pasa al amarillo:

background-image: linear-gradient(to right, red, yellow);

Dirección - Diagonal

Puede realizar un degradado en diagonal especificando las posiciones iniciales horizontales y verticales.

El siguiente ejemplo muestra un degradado lineal que comienza en la parte superior izquierda (y continúa hasta la parte inferior derecha). Comienza en rojo y pasa a amarillo:

background-image: linear-gradient(to bottom right, red, yellow);

Usando ángulos

Si desea tener más control sobre la dirección del gradiente, puede definir un ángulo en lugar de las direcciones predefinidas (hacia abajo, hacia arriba, hacia la derecha, hacia la izquierda, hacia abajo a la derecha, etc.). Un valor de 0 grados equivale a "hacia arriba". Un valor de 90 grados equivale a "hacia la derecha". Un valor de 180 grados equivale a "hacia abajo".

Sintaxis

background-image: linear-gradient(angle, color-stop1, color-stop2);

El siguiente ejemplo muestra cómo utilizar ángulos en gradientes lineales:

background-image: linear-gradient(180deg, red, yellow);

Uso de múltiples paradas de color

El siguiente ejemplo muestra un degradado lineal (de arriba a abajo) con múltiples paradas de color:

background-image: linear-gradient(red, yellow, green);

El siguiente ejemplo muestra cómo crear un degradado lineal (de izquierda a derecha) con el color del arco iris y algo de texto:

background-image: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);

Usando Transparencia

Los degradados CSS también admiten transparencia, que puede utilizarse para crear efectos de desvanecimiento.

Para agregar transparencia, utilizamos la función rgba() para definir los niveles de color. El último parámetro de la función rgba() puede ser un valor de 0 a 1 y define la transparencia del color: 0 indica transparencia total, 1 indica color total (sin transparencia).

El siguiente ejemplo muestra un degradado lineal que comienza desde la izquierda. Comienza totalmente transparente y pasa a ser completamente rojo:

background-image: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1));

Repetición de un gradiente lineal

La función repeating-linear-gradient() se utiliza para repetir gradientes lineales:

CÓDIGO:

background-image: repeating-linear-gradient(red, yellow 10%, green 20%);

Degradados radiales

Un gradiente radial se define por su centro.

Para crear un degradado radial también debes definir al menos dos paradas de color.

Sintaxis

background-image: radial-gradient(shape size at position, start-color, ..., last-color);

De forma predeterminada, la forma es elipse, el tamaño es la esquina más alejada y la posición es el centro.

Degradado radial: paradas de color espaciadas uniformemente (esta opción es predeterminada)

El siguiente ejemplo muestra un degradado radial con paradas de color espaciadas uniformemente:

background-image: radial-gradient(red, yellow, green);

Degradado radial: paradas de color con diferentes espacios

El siguiente ejemplo muestra un degradado radial con paradas de color espaciadas de forma diferente:

background-image: radial-gradient(red 5%, yellow 15%, green 60%);

Gradientes cónicos

Un degradado cónico es un degradado con transiciones de color rotadas alrededor de un punto central.

Para crear un degradado cónico debes definir al menos dos colores.

Sintaxis

background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...);

De forma predeterminada, el ángulo es 0 grados y la posición es el centro.

Si no se especifica ningún grado , los colores se distribuirán uniformemente alrededor del punto central.

Gradiente cónico: tres colores

El siguiente ejemplo muestra un gradiente cónico con tres colores:

background-image: conic-gradient(red, yellow, green);

Gradiente cónico: cinco colores

El siguiente ejemplo muestra un gradiente cónico con cinco colores:

background-image: conic-gradient(red, yellow, green, blue, black);

Gradiente cónico: tres colores y grados

El siguiente ejemplo muestra un gradiente cónico con tres colores y un grado para cada color:

background-image: conic-gradient(red 45deg, yellow 90deg, green 210deg);

Crear gráficos circulares

Solo agréguelo border-radius: 50% para hacer que el gradiente cónico parezca un pastel:

CÓDIGO:

background-image: conic-gradient(red, yellow, green, blue, black);

border-radius: 50%;

Aquí hay otro gráfico circular con grados definidos para todos los colores:

CÓDIGO:

background-image: conic-gradient(red 0deg, red 90deg, yellow 90deg, yellow 180deg, green 180deg, green 270deg, blue 270deg);

border-radius: 50%;

Repetición de un gradiente cónico

La función repeating-conic-gradient() se utiliza para repetir gradientes cónicos:

CÓDIGO:

background-image: repeating-conic-gradient(red 10%, yellow 20%);

border-radius: 50%;

Aquí se muestra un gradiente cónico repetitivo con inicios y finales de color definidos:

CÓDIGO:

background-image: repeating-conic-gradient(red 0deg 10deg, yellow 10deg 20deg, blue 20deg 30deg);

border-radius: 50%;

CSS Shadow

Con CSS puedes agregar sombra al texto y a los elementos.

En estos capítulos aprenderás sobre las siguientes propiedades:

  • text-shadow
  • box-shadow

Sombra de texto (text-shadow)

La propiedad text-shadow aplica sombra al texto.

En su uso más simple, solo especifica la sombra horizontal (2px) y la sombra vertical (2px), pero también le puedes poner colores:

¡Efecto de sombra de texto!

CÓDIGO:

text-shadow: 2px 2px red;

Luego, agrega un efecto de desenfoque a la sombra:

¡Efecto de sombra de texto!

CÓDIGO:

text-shadow: 2px 2px 5px red;

Sombras múltiples

Para agregar más de una sombra al texto, puede agregar una lista de sombras separadas por comas.

El siguiente ejemplo muestra un texto blanco con sombras negras, rojo y rojo oscuro:

¡Efecto de sombra de texto!

CÓDIGO:

text-shadow: 1px 1px 2px black, 0 0 25px red, 0 0 5px rgb(148, 0, 0);

También puedes usar la propiedad text-shadow para crear un borde simple alrededor de algún texto (sin sombras):

¡Borde alrededor del texto!

CÓDIGO:

text-shadow: -1px 0 black, 0 1px black, 1px 0 black, 0 -1px black;

Sombra de cuadro (box-shadow)

La propiedad box-shadow se utiliza para aplicar una o más sombras a un elemento.

Especificar una sombra horizontal y una vertical

En su uso más simple, solo se especifica una sombra horizontal y una vertical. El color predeterminado de la sombra es el color del texto actual, pero se le puede añadir color.

box-shadow: 5px 10px gray;

Para añadir desenfoque a la sombra usamos el parámero blur. Cuanto mayor sea el número, más borrosa será la sombra.

box-shadow: 10px 10px 5px gray;

Agregar múltiples sombras

Un elemento también puede tener múltiples sombras:

box-shadow: 5px 5px blue, 10px 10px red, 15px 15px green;

CSS Web Fonts

Las fuentes web permiten a los diseñadores web utilizar fuentes que no están instaladas en la computadora del usuario.

Cuando haya encontrado/comprado la fuente que desea utilizar, simplemente incluya el archivo de fuente en su servidor web y se descargará automáticamente para el usuario cuando sea necesario.

Tus fuentes "propias" están definidas dentro de la regla @font-face.

Diferentes formatos de fuentes

Fuentes TrueType (TTF)

TrueType es un estándar de fuentes desarrollado a finales de la década de 1980 por Apple y Microsoft. TrueType es el formato de fuente más común tanto para los sistemas operativos Mac OS como para Microsoft Windows.

El formato de fuente abierto para la Web (WOFF)

WOFF es un formato de fuente para uso en páginas web. Fue desarrollado en 2009 y ahora es una recomendación del W3C. WOFF es esencialmente OpenType o TrueType con compresión y metadatos adicionales. El objetivo es permitir la distribución de fuentes desde un servidor a un cliente a través de una red con restricciones de ancho de banda.

El formato de fuente abierto para la Web (WOFF 2.0)

Fuente TrueType/OpenType que proporciona una mejor compresión que WOFF 1.0.

Fuentes/formas SVG

Las fuentes SVG permiten utilizar SVG como glifos al mostrar texto. La especificación SVG 1.1 define un módulo de fuentes que permite la creación de fuentes dentro de un documento SVG. También se pueden aplicar CSS a documentos SVG y la regla @font-face se puede aplicar al texto en documentos SVG.

Fuentes OpenType integradas (EOT)

Las fuentes EOT son una forma compacta de fuentes OpenType diseñadas por Microsoft para su uso como fuentes integradas en páginas web.

Usando la fuente que quieras

En la regla @font-face primero defina un nombre para la fuente (por ejemplo, Relationship of Melodrame) y luego apunte al archivo de fuente.

Para utilizar la fuente para un elemento HTML, haga referencia al nombre de la fuente (Relationship of Melodrame) a través de la propiedad font-family:

font-family: Relationship of Melodrame;

CÓDIGO:

En @font-face:

font-family: 'Relationship of Melodrame';

src: url(fondos/relationship_of_melodrame/Relationship\ of\ mélodrame.ttf);

En p:

font-family: 'Relationship of Melodrame';

Uso de texto en negrita

Se usa @font-weight para ajustar el grosor de nuestro texto, normalmente se usa el valor bold.

font-family: Relationship of Melodrame;

CÓDIGO:

font-weight: bold;

CSS Box Sizing

La propiedad box-sizing nos permite incluir el relleno y el borde en el ancho y alto total de un elemento.

Sin la propiedad box-sizing

De forma predeterminada, el ancho y la altura de un elemento se calculan de la siguiente manera:

ancho + relleno + borde = ancho real de un elemento

alto + relleno + borde = alto real de un elemento

Esto significa: cuando establece el ancho/alto de un elemento, el elemento a menudo aparece más grande que lo establecido (porque el borde y el relleno del elemento se agregan al ancho/alto especificado del elemento).

La siguiente ilustración muestra dos elementos "div" con el mismo ancho y alto especificados:

width: 300px;

height: 100px;

border: 1px solid blue;

width: 300px;

height: 100px;

border: 1px solid red;

padding: 30px;

CÓDIGO:

resize: both;

overflow: auto;

Los dos elementos "div" anteriores terminan con tamaños diferentes en el resultado (porque el segundo "div" tiene un relleno especificado).

La propiedad box-sizing resuelve este problema.

Con la propiedad box-sizing

La propiedad box-sizing nos permite incluir el relleno y el borde en el ancho y alto total de un elemento.

Si configuramos box-sizing: border-box; en un elemento, el relleno y el borde se incluyen en el ancho y la altura:

width: 300px;

height: 150px;

border: 1px solid blue;

box-sizing: border-box;

width: 300px;

height: 150px;

border: 1px solid red;

padding: 30px;

box-sizing: border-box;

CÓDIGO:

box-sizing: border-box;
